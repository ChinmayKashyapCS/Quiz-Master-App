# 📘 Quiz Master App

This is a full-stack **Quiz Management Web Application** built using modern frontend and backend technologies. The application enables administrators to create and manage quizzes, while users can register, attempt quizzes, and track their performance.

---

## 🧠 Overview

The Quiz Master App is designed to provide an engaging quiz experience through a structured system of subjects, chapters, and quizzes. It supports secure user authentication, role-based access (admin and user), quiz participation, score tracking, and responsive UI interaction.

## 🚀 Features

### 🧑‍💻 User Features

- Secure **user registration and login**
- Participate in quizzes by subject and chapter
- View past quiz performance and scores

### 👨‍🏫 Admin Features

- Create and manage **subjects, chapters, quizzes, and questions**
- View quiz details and participant performance

---

## 🛠️ Technologies Used

- **Backend:** Python, Flask
- **Frontend:** Vue.js
- **Database:** (e.g., SQLite/PostgreSQL)
- **RESTful API** for frontend-backend communication
- **JWT authentication**
